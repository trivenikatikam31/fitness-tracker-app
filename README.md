# fitness-tracker-app
Fitness Tracker App is a responsive web application built with React and SQLite that enables users to record daily fitness activities, track steps, monitor workout duration, calculate calories burned, and analyze performance through interactive dashboards, charts, and progress indicators.
# 🏃 Fitness Tracker App

A modern and user-friendly Fitness Tracker App that helps users monitor their daily fitness activities, track workouts, record calories burned, and visualize progress through interactive dashboards and charts.

## 🚀 Features

### Dashboard
- Daily fitness summary
- Total steps tracked
- Calories burned
- Workout duration
- Progress indicators
- Weekly analytics charts

### Activity Management
- Add fitness activities
- Edit activity records
- Delete activity records
- View activity history

### Activity Types
- Walking
- Running
- Cycling
- Gym Workouts
- Yoga
- Custom Activities

### Analytics
- Daily progress tracking
- Weekly performance charts
- Activity trends
- Goal completion percentage

### Additional Features
- Responsive Design
- Dark Mode Support
- Search & Filter Activities
- CSV Export Functionality
- SQLite Database Storage

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/fitness-tracker-app.git
```

### Navigate to Project

```bash
cd fitness-tracker-app
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
